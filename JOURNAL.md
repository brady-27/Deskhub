# 5/22/2026 10 PM - 3d Model

_Time spent: 3.45h_

First, I decided on the general shape of the deviceand decided that I wanted the screen to be on an angle for better ergonomic usage. I measured the space that I had for it under my monitor and put it in Fusion360 and started designing from there. 

Eventually I decided that I wanted the back of the plate to be reusable so I added holes to add magnets mid print into the main shell and the back plate. 


After this, I found the parts that I wanted to use and decided on the Waveshare ESP32-S3-Touch-LCD-7 Inch Capacitive Touch Display that also has a ESP32 which is great cause it saves space and money. After this I found the sensor that I wanted to use; an Infrared PIR Motion Sensor.
