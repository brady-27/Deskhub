# Deskhub
A small smart desktop touchscreen that displays time, date, weather, Spotify, and a to-do list. Users can swipe between pages, control Spotify with on-screen buttons and a volume slider, and the device includes RGB backlighting plus a motion sensor to wake the screen when nearby.

I got this idea originally off of my friend who had a matric led display jsut dispaying cool things like a clock so I wanted something like that but able to show TIme, weather, date, and even spotify.
